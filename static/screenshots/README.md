# Screenshots Directory

Place screenshots of supported mods here to display them on the home page.

## Required Files

- `mistwarp.png` - Screenshot of MistWarp showing a themed instance (recommended: 1200x900px)
- `nitrobolt.png` - Screenshot of NitroBolt showing a themed instance (recommended: 1200x900px)

## Guidelines

- Screenshots should showcase themes using WarpTheme
- Show the full interface or meaningful sections
- Use high-quality, clear images
- PNG or JPG format recommended
- Maintain consistent aspect ratio if possible
